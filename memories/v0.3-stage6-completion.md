interface MediaCategorySelectProps {

- 展示分类徽标（图标+名称）

## 技术决策

### 为什么选择这个方案
1. **最小化改动**: 只在 Upload 页面接入，不涉及其他页面
2. **可复用性**: 两个组件可被其他页面复用
3. **渐进式**: 为后续 Tauri 集成和 Theme Engine 预留接口
4. **类型安全**: 完整的 TypeScript 支持

### 避免的过度设计
- ❌ 不创建全局 media context（暂时不需要）
- ❌ 不做分类权限管理（当前无此需求）
- ❌ 不做分类排序/拖拽（MVP 阶段不需要）
