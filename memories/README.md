# InkForge 记忆库索引

> 本目录是 InkForge 的治理层文档，负责让协作入口足够快，让状态、决策、执行边界不再漂移。

---

## 会话启动必读

每次新会话或进入新任务时，按以下顺序读取：

1. `../CODEBUDDY.md`
2. `PROJECT_STATUS.md`
3. `RUNTIME_RULES.md`

---

## 文档职责

| 文档 | 职责 |
|------|------|
| `PROJECT_STATUS.md` | 当前真实状态、已验证能力、技术债清单、近期优先级 |
| `architecture-decisions.md` | 架构决策记录 + 可执行路线图 |
| `RUNTIME_RULES.md` | 运行规则 + 工程质量底线 |
