**创建时间**: 2026-04-06
**状态**: 规划中
**优先级**: P1

## 目标
打通前台主题渲染、静态资源与最小预览链路，使主题从“后台可切换”提升为“前台可生效”。

## 当前现状
- `router.rs` 中前台主题渲染路由仍是 TODO。
- 后端已有 active theme / list / activate 基础能力。
- 主题模板目录已存在：`themes/default/templates/*.html`。
- 当前主题系统主要停留在后台管理层，尚未形成运行时闭环。

## 迭代 1：前台主题渲染入口
- 启用首页、文章页、资料页、静态资源路由

- 建立模板加载入口

## 迭代 2：最小预览能力
- 提供预览数据接口
- 支持以 `theme_slug` 方式预览而不修改 active theme
- 只覆盖首页/文章页最小场景
