# InkForge 应用图标

## 图标来源

所有图标由 `src/app-icon.svg` 转换而来。

## 需要的文件

请将以下 PNG 文件放置在此目录：

- [ ] `32x32.png` - 32x32 像素
- [ ] `128x128.png` - 128x128 像素
- [ ] `128x128@2x.png` - 256x256 像素（Retina）
- [ ] `icon.png` - 512x512 或 1024x1024 像素

## 转换步骤

1. 访问 https://svg-to-png.org/
2. 上传 `../../src/app-icon.svg`
3. 生成上述尺寸的 PNG 文件
4. 保存到此目录

详细指南请查看 `../../docs/ICON_GUIDE.md`
